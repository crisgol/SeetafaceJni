# SeetafaceJni
  根据Seetaface6相关文件，尝试自己编译Android可以调用的Jni文件
## 前言
这是自己第一个Jni项目，第一次编译ndk相关文件

## SeetafaceJni介绍
在使用seetaface6过程中，需要使用人脸追踪还有图像检测功能，但是原文件提供的demo并没有可供Android 调用的_java.so，并且我看网上大家都在找v86a的包，所以我想自己尝试研究编译出来
